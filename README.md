# Adalin

LINbus 2.2 package written in Ada. This repository contains the core `Adalin` library and a small AUnit test suite.

Quick start

Prerequisites
- GNAT (Ada compiler) installed
- Alire (Ada package manager / build tool)

Build
```
alr build
```

Run tests
```
alr test
```

Tests
- Test sources are under `tests/` (e.g. `tests/tests_adalin_frame.adb`).
